# coc-project-calculator
its a simple calculator created in c language.

c: 1) #include<math.h> for predefine math function

   2) #include<ctype.h> to use lower function.
    
   3) function, while,if statement, switch,relational operator.
   
math: additiion,subtraction,multiplication,division,modulou,factorial,power,root,logarithm,trignometric function,utilities like max,min,ceil,floor,abs.


how to compile : gcc calculator.c -o calculator -lm


how to run: ./calculator
